> **游戏说明:**

> - 使用鼠标控制
> - 吃**钻石**获得特殊技能
> - 躲避**除了钻石以外**的飞行物体~
> - 建议戴上耳机

> **特色:**

> - 几乎所有元素都为js动态生成
> - **全部**图形绘制由js代码完成，未使用任何外部图片
